We as a group,have connected an computer which has software that holds the code and executes through the input given by the user. To this computer we connect our hardware model through USB .In hardware model we have an ardino chip which is connected to 2 motor drives to which 3 motor rollers are connected.
 This hardware model have 2 linear shafts which move in x and y directions on each execution.There is an pen holder in which a pen is kept to prints in braille script according to the input given by the user.Our main theme of the project is to educate blind people through braille script.
